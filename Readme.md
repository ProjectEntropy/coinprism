Talk to the coin prism blockchain explorer and wallet API

## Quick start

Install via Rubygems

    gem install coinprism

... or add to your Gemfile

    gem "coinprism"


### Making requests
```ruby
Coinprism.address_info("akW5d2jL859ELCxvRwtqiiZV8Hg5Bx7Pu19")


```
